# Raipur Dust Dashboard

Simple static dashboard for Raipur air quality / dust monitoring.

How to run locally
- Open [index.html](index.html) in a browser.

Linter
- Requires Node.js. Run:

```bash
npm install
npm run lint
```

CI
- A GitHub Actions workflow runs `npm install` and `npm run lint` on push/pull requests.
